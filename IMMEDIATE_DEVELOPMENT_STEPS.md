## 🔧 开发步骤

### 步骤1: 打开LobbyScene
```
操作:
1. 在Cocos Creator资源管理器中
2. 导航到: assets/scenes/
3. 双击: LobbyScene.scene
4. 场景编辑器中将打开空场景

预期:
✅ 场景编辑器显示LobbyScene
✅ 层级管理器显示空场景
```

### 步骤2: 添加Canvas节点
```
操作:
1. 在层级管理器中，右键点击"场景"节点
2. 选择"创建节点" → "创建渲染节点" → "Canvas"
3. 重命名为: "Canvas"

属性设置:
- Design Resolution: 宽度 1920, 高度 1080
- Fit Height: 勾选 ✅
- Fit Width: 勾选 ✅

预期:
✅ Canvas节点出现在层级管理器中
✅ 属性检查器显示Canvas设置
```

### 步骤3: 添加背景
```
操作:
1. 在层级管理器中，右键点击Canvas节点
2. 选择"创建节点" → "创建渲染节点" → "Sprite"
3. 重命名为: "Background"

属性设置:
- Position: X=0, Y=0
- Size: 宽度 1920, 高度 1080
- Color: R=30, G=30, B=60, A=255 (深蓝色)

预期:
✅ 背景出现在场景中
✅ 屏幕显示深蓝色背景
```

### 步骤4: 添加游戏标题
```
操作:
1. 在Canvas下右键 → "创建节点" → "创建UI节点" → "Label"
2. 重命名为: "TitleLabel"

属性设置:
- Position: X=0, Y=300
- String: "赛马游戏大厅"
- FontSize: 72
- Color: R=255, G=255, B=0, A=255 (黄色)
- HorizontalAlign: CENTER

预期:
✅ 屏幕顶部显示黄色标题
```

### 步骤5: 添加欢迎信息
```
操作:
1. 创建Label节点
2. 重命名为: "WelcomeLabel"

属性设置:
- Position: X=0, Y=200
- String: "欢迎来到赛马游戏!"
- FontSize: 36
- Color: R=255, G=255, B=255, A=255 (白色)
- HorizontalAlign: CENTER

预期:
✅ 标题下方显示白色欢迎信息
```

### 步骤6: 添加开始游戏按钮
```
操作:
1. 创建Button节点
2. 重命名为: "StartButton"

属性设置:
- Position: X=0, Y=0
- Size: 宽度 300, 高度 100
- Color: R=0, G=150, B=255, A=255 (蓝色)

添加按钮文字:
1. 在StartButton节点上右键 → "创建节点" → "创建UI节点" → "Label"
2. 重命名为: "StartButtonLabel"
3. 属性设置:
   - String: "开始游戏"
   - FontSize: 36
   - Color: R=255, G=255, B=255, A=255 (白色)
   - HorizontalAlign: CENTER
   - VerticalAlign: MIDDLE

预期:
✅ 屏幕中央显示蓝色开始按钮
✅ 按钮上有白色文字"开始游戏"
```

### 步骤7: 添加设置按钮
```
操作:
1. 创建Button节点
2. 重命名为: "SettingsButton"

属性设置:
- Position: X=0, Y=-120
- Size: 宽度 300, 高度 100
- Color: R=100, G=100, B=100, A=255 (灰色)

添加按钮文字:
1. 创建Label子节点
2. 重命名为: "SettingsButtonLabel"
3. 属性设置:
   - String: "设置"
   - FontSize: 36
   - Color: R=255, G=255, B=255, A=255

预期:
✅ 开始按钮下方显示灰色设置按钮
```

### 步骤8: 添加退出按钮
```
操作:
1. 创建Button节点
2. 重命名为: "ExitButton"

属性设置:
- Position: X=0, Y=-240
- Size: 宽度 300, 高度 100
- Color: R=255, G=50, B=50, A=255 (红色)

添加按钮文字:
1. 创建Label子节点
2. 重命名为: "ExitButtonLabel"
3. 属性设置:
   - String: "退出游戏"
   - FontSize: 36
   - Color: R=255, G=255, B=255, A=255

预期:
✅ 设置按钮下方显示红色退出按钮
```

### 步骤9: 添加LobbyUI脚本
```
操作:
1. 在层级管理器中选中Canvas节点
2. 在属性检查器中，点击"添加组件"按钮
3. 选择"用户脚本组件" → "LobbyUI"

预期:
✅ Canvas节点添加了LobbyUI组件
✅ 属性检查器显示LobbyUI属性
```

### 步骤10: 连接UI引用
```
操作:
在LobbyUI组件属性中:
1. welcomeLabel: 拖拽WelcomeLabel节点到该属性框
2. scoreLabel: 拖拽一个空的Label节点（暂时不用）
3. startButton: 拖拽StartButton节点
4. settingsButton: 拖拽SettingsButton节点
5. exitButton: 拖拽ExitButton节点

技巧:
- 从层级管理器拖拽节点到属性框
- 确保每个属性都连接了正确的节点

预期:
✅ 所有UI引用连接完成
✅ 属性框中显示节点名称
```

### 步骤11: 保存场景
```
操作:
1. 菜单栏 → 文件 → 保存场景
2. 或按快捷键: Ctrl+S

预期:
✅ 场景保存成功
✅ 没有错误提示
```

### 步骤12: 设置启动场景
```
操作:
1. 菜单栏 → 项目 → 项目设置
2. 选择"启动场景"标签
3. 点击"选择"按钮
4. 选择: LobbyScene

预期:
✅ LobbyScene设置为启动场景
✅ 下次运行游戏时自动打开大厅
```

## 🔧 故障排除

### 问题1: UI元素不显示
```
可能原因:
1. Canvas设置不正确
2. 节点不在Canvas下
3. Position设置错误

解决:
1. 检查Canvas的Design Resolution
2. 确认节点层级关系
3. 调整Position值
```

### 问题2: 按钮不响应
```
可能原因:
1. LobbyUI脚本未挂载
2. UI引用未连接
3. 按钮Interactable为false

解决:
1. 重新挂载LobbyUI脚本
2. 重新连接UI引用
3. 检查按钮属性
```

### 问题3: 脚本编译错误
```
可能原因:
1. TypeScript语法错误
2. 导入路径错误
3. 缺少依赖

解决:
1. 查看控制台错误信息
2. 检查脚本语法
3. 确保所有文件存在
```
