# 资源占位文件说明

此目录用于存放游戏资源文件。请将相应的资源文件放入对应的子目录。

## 目录结构说明

### 图片资源
- horses/ - 赛马图片 (建议尺寸: 128x128像素)
- ui/buttons/ - 按钮图片
- ui/icons/ - 图标图片  
- ui/backgrounds/ - 背景图片

### 音效资源
- audio/bgm/ - 背景音乐 (MP3格式)
- audio/sfx/ - 音效文件 (WAV/MP3格式)

### 字体资源
- fonts/ - 游戏字体文件 (TTF/OTF格式)

### 其他资源
- animations/ - 动画文件
- prefabs/ - 预制体文件
- scenes/ - 场景文件

## 资源要求
1. 图片格式: PNG (支持透明) 或 JPG
2. 音效格式: MP3 或 WAV
3. 命名规范: 使用英文小写和下划线，如: horse_red.png
4. 尺寸规范: 遵循2的幂次方 (64, 128, 256, 512等)

请将您的游戏资源文件复制到对应的目录中，然后更新资源引用。
