## 🔧 关键操作步骤

### 1. 添加Canvas
```
右键场景 → 创建节点 → 创建渲染节点 → Canvas
设置: 1920x1080, Fit Height/Width
```

### 2. 添加UI元素
```
在Canvas下右键 → 创建节点 → 选择类型
类型: Sprite(背景), Label(文字), Button(按钮)
```

### 3. 挂载脚本
```
选中Canvas → 属性检查器 → 添加组件 → 用户脚本组件 → LobbyUI
```

### 4. 连接引用
```
从层级管理器拖拽节点到LobbyUI属性框
需要连接: welcomeLabel, startButton, settingsButton, exitButton
```

### 5. 保存测试
```
Ctrl+S保存 → Ctrl+P运行
检查: 按钮点击 → 控制台日志
```

## ⚠️ 常见问题

### UI不显示？
```
检查: Canvas设置, 节点层级, Position值
```

### 按钮不响应？
```
检查: LobbyUI是否挂载, 引用是否连接
```

### 编译错误？
```
查看控制台错误信息, 检查TypeScript语法
```
